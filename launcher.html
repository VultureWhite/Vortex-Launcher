What's New
Changes to Vortex Launcher itself — not Minecraft.
v2.4.0 Latest
  • Modpacks are now a separate global page — downloading one automatically creates and configures a whole new instance for you, no picking an instance required.
  • Each instance has its own Versions tab now, so you can view or change its Minecraft version and loader without leaving the instance.
  • Per-instance Downloads now covers Mods, Resource Packs, Shader Packs and Data Packs.
  • You can upload a custom image as an instance's icon, both when creating it and afterwards in its Settings tab.
  • Saves and Logs tabs are alive — add worlds to an instance's Saves list, and launching now writes real entries into its Logs.
v2.3.0
  • Introduced the instance system: create named instances, each with its own version, loader, icon, mods, resource packs, shader packs, saves and settings.
  • The Play bar is now pinned to the bottom across every page and gated on having an instance selected.
  • Removed the old global Downloads page in favor of per-instance downloads.
v2.2.0
  • Added a skin selector with presets and custom PNG upload, rendered as a front-facing paper-doll preview.
  • Added light/dark theme switching and a proper slide-in menu for narrow screens.
